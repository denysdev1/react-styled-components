## Run Locally

Clone the project

```bash
  git clone git@github.com:denysdev1/react-mui-project.git
```

Go to the project directory

```bash
  cd react-mui-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
